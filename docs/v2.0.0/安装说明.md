# Why App 安装说明

1. 将 `Why-v2.0.0-debug.apk` 发送到 Android 手机。
2. 在手机文件管理器中点击 APK。
3. 如果系统提示，请允许"安装未知来源应用"。
4. 完成安装后，桌面应用名称为 `Why`。

## 兼容范围

- Android 7.0 及以上
- 排行榜需要网络；练习和本机题量记录可离线使用
- 只申请联网权限，不读取通讯录、位置或相册
